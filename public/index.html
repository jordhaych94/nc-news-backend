**NC News API**

The backend data service for NC News, enabling full access to articles, topics, and social interactions.

GET POST PUT PATCH DELETE
GET POST PUT PATCH DELETE

**Available Endpoints:**

  • GET /api/topics

    Retrieve a list of all topics.

    Example response:

                
          {
            "topics": [
              {
                "slug": "coding",
                "description": "Code is love, code is life"
              },
              {
                "slug": "football",
                "description": "FOOTIE!"
              },
              {
                "slug": "cooking",
                "description": "Hey good looking, what you got cooking?"
              }
            ]
          }
