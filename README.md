# NC News Seeding

- Hello from README
- instructions/setup

1. install packages needed with ' npm install '.
2. create .env.test && .env.developtment files.
3. in .env.test make sure you write ' PGDATABASE=database_name_test ' i.e ' PGDATABASE=nc_news_test '.
4. in .env.development make sure you write ' PGDATABASE=database_name ' i.e ' PGDATABASE=nc_news '.
